Как разместить сайт:

  1. Скопируйте весь код этой страницы (нажмите Ctrl+A затем Ctrl+C)
  2. Откройте Replit.com в браузере
  3. Создайте новый проект (выберите "HTML, CSS, JS")
  4. Замените содержимое файла index.html на скопированный код
  5. Нажмите кнопку "Run" вверху
  6. Скопируйте URL из адресной строки браузера
  7. Поделитесь этой ссылкой с другими

Теперь ваш сайт доступен по постоянной ссылке!
